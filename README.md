# Bee Deals AI Product Description Generator

An AI-powered web app that scrapes product information from a URL, HTML, or text and generates high-quality, customizable product descriptions, headers, feature lists, and review summaries for eCommerce.

##  Features

-   **Multi-Source Scraping:** Paste a product URL, raw HTML, or plain text to get started.
-   **AI-Powered Content Generation:** Automatically creates a product photo, catchy header, engaging description, and a clean features list.
-   **Review Summarization:** If customer reviews are found, the AI summarizes them into positive and negative points.
-   **Deep Customization:**
    -   **Tone:** Adjust the writing style (Formal, Casual, Persuasive).
    -   **Length:** Control the description's word count (Short, Medium, Long, or Auto).
    -   **Emojis:** Choose to include or exclude emojis.
    -   **Language:** Generate and translate content between English and Hebrew.
-   **Interactive Editor:** Edit any generated content directly in the app.
-   **Export Options:** Copy content as Markdown/HTML or download a complete HTML file.

##  How to Use

1.  **Input:** Paste a product URL, HTML code, or product text into the main input box.
2.  **Customize:** Use the sidebar to set your desired tone, length, and other options.
3.  **Generate:** Click the "Generate" button.
4.  **Edit & Export:** Refine the generated content as needed and use the copy/download buttons to export your results.
